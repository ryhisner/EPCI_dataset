These are lists of EPI_ISL numbers for EPCI sequences. 
The "all_EPCI_sequences" file includes repeat sequences from the same patient (or assumed to be from the same patient.
The "independent_qualifying_sequences" includes only sequences not closely related to any other sequences on the list. 
For groups of related sequences (presumed or known to be from the same patient), the sequence with the highest number of private AA mutations is selected as a single representative.

In dozens of these sequences, artifactual frameshifts had to be manually fixed (e.g. by deleting or inserting 'N' nucleotides) in order for Nextclade analysis to be possible.
